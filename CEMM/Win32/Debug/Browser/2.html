Description
A simple Mod which accelerates the Elevators.

Vertical Elevator 5x Speed
Horizontal Elevator 2.5x Speed


This Mod modifies the Entries for the Elevators in the BP_PL_Elevator, BP_PL_Elevator_Horizontal and should be compatible with all Mods that do not modify the same Blueprints.
